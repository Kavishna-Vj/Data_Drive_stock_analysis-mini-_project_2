Data Drive Stock Analysis:

• The data drive stock analysis is a process of collecting, cleaning, transforming and analysing stock markets.

Technologies Used:

⁕ Python
⁕ Pandas
⁕ Matplotlib/Seaborn
⁕ MYSQL
⁕ YAML converting or processing
⁕ File handling(folder wise , symbol wise)
⁕ steamlit

Key Features:

⁕ Convert raw YAML files into CSV files
⁕ Organizing data as ;
   → Symbol wise folder
   → Monthly return folder
   → Analysis the output
   → Clean and preprocess stock data

 Database Connection:
   ⁕ To use mysql-connector python for long term access

 Visulization:
  ⁕ Cumulative return
  ⁕ Volatile return
  ⁕ Top 10 Gainers and Losers
  ⁕ Monthly return
  ⁕ Top 5 & Bottom 5 monthly loser and gainer
  ⁕ Correlation (Heatmap) , the streamlit dashboard can be used visualise the followings.
